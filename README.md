# MD5加密工具 - PyQt6实现

## 项目简介

一个基于PyQt6图形界面开发的MD5加密工具，提供简洁美观的用户界面，支持实时字符统计和大小写转换功能。适合学习PyQt6开发、密码学基础应用以及日常简单的加密需求。

## 功能特性

### 🔧 核心功能
- **MD5加密**：将输入文本快速转换为32位MD5哈希值
- **大小写切换**：支持加密结果的大写/小写格式输出
- **实时统计**：实时显示输入字符数量
- **一键复制**：快速复制加密结果到剪贴板
- **数据清空**：一键清空输入和输出区域

### 🎨 界面特点
- 响应式布局设计，支持窗口大小调整
- 状态栏实时反馈操作结果
- 简洁直观的用户界面
- 完善的工具提示（ToolTip）

## 运行环境

### 系统要求
- Windows 7/8/10/11, macOS, Linux
- Python 3.8 或更高版本

### 依赖库
```bash
PyQt6 >= 6.0.0
```

## 安装与使用

### 安装方法
1. 克隆或下载本项目
2. 安装依赖库：
```bash
pip install PyQt6
```

### 运行程序
```bash
python md5_enc.py
```

### 使用步骤
1. 在输入框中输入需要加密的文本
2. （可选）勾选"输出大写"以获取大写的MD5结果
3. 点击"加密"按钮生成MD5值
4. 点击"复制"将结果复制到剪贴板
5. 点击"清空"重置所有内容

### 代码模块说明
- `set_widget_text()`: 统一的字体样式设置函数
- `Md5Window`类: 主窗口类，包含所有UI组件和功能逻辑
- 四个主要区域：
  - 输入区域：文本输入框
  - 控制区域：选项和统计信息
  - 输出区域：加密结果显示
  - 按钮区域：操作按钮

## 技术要点

### 核心技术
1. **PyQt6应用架构**
   - 基于QMainWindow的主窗口
   - QVBoxLayout/QHBoxLayout布局管理
   - 信号与槽机制的事件处理

2. **MD5加密实现**
   - 使用Python标准库`hashlib`
   - UTF-8编码处理
   - 十六进制字符串输出

3. **UI优化**
   - 自适应窗口大小
   - 统一的字体样式管理
   - 完善的用户反馈机制

### 关键函数
- `encrypt_text()`: 执行MD5加密
- `update_char_count()`: 实时更新字符计数
- `is_uper()`: 大小写转换处理
- `copy_text()`: 剪贴板操作

## 注意事项

1. **编码问题**：程序使用UTF-8编码，确保输入文本编码正确
2. **安全性说明**：MD5是单向哈希函数，适用于简单的数据验证，不建议用于密码存储等安全要求高的场景
